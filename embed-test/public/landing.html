Integration Guide

1. Add the Widget Script

<script>
  window.OzwellChatConfig = {
    endpoint: '/mock/chat',
    widgetUrl: '/embed/widget.html'
  };
</script>
<script src="/embed/embed.js"></script>

2. Define MCP Tools

window.OzwellChatConfig = {
  ...,
  tools: [{
    name: 'update_name',
    description: 'Updates user name',
    parameters: {
      type: 'object',
      properties: {
        name: { type: 'string' }
      },
      required: ['name']
    }
  }]
};

3. Handle Tool Calls

window.addEventListener('message', (e) => {
  if (e.data.type === 'tool_call') {
    // Update your form/UI
    nameInput.value = e.data.payload.name;
  }
});

How It Works

  • Parent page defines tools and handlers
  • Widget receives context via iframe-sync
  • LLM calls tools based on user intent
  • Tool calls sent via postMessage to parent
  • Parent executes handlers and updates UI
